# 变更日志 (Changelog)

所有关于 Ledom Layouter 的显著更改都将记录在此文件中。
本项目遵循 [语义化版本 2.0.0](https://semver.org/lang/zh-CN/) 规范。

## [1.3.0] - 2026-04-09
### 🚀 新增 (Added)
- 引入了 `run_layouter.bat` 极客风启动脚本，支持 Python 环境与依赖库的自动化自检与静默修复。
- 在编辑器侧边栏新增 `ui_height` 动态高度调节滑块，支持大屏及远程桌面场景下的视口自适应。
- 在转换历史面板新增“清空历史”按钮，完善数据管理闭环。

### 🔄 优化 (Changed)
- 升级文件生命周期管理，将导出文件数据持久化至缓存，彻底解决下载按钮点击后“闪退消失”的生命周期 Bug。
- 优化前端交互状态机，在执行 Pandoc 转换期间自动锁定导出按钮，防止高频点击导致的进程卡死。
- 滑块调节高度时接入 `st.rerun()` 实现即时渲染刷新。

### 🛠 修复 (Fixed)
- **[Critical]** 修复了组件初始化时序问题导致的 `NameError`，确保 CSS 能够安全注入动态变量。
- **[Critical]** 修复了由于 Markdown 代码块反引号 (``````) 导致的 Streamlit 前端渲染器“逃逸截断”漏洞，通过动态字符生成（`chr(96) * 3`）保障了示例代码的完整闭合。
- 优化了 CSS 补丁，彻底隐藏 `text_area` 残留占位，实现编辑器与预览框像素级顶部对齐。

---

## [1.2.3] - 2026-03
### 🚀 新增 (Added)
- 项目立项，确立“Local-First (本地优先)”架构。
- 集成 Pandoc 核心转换引擎，打通 Markdown 到 Word 的 `--reference-doc` 样式映射通道。
- 实现基于剪贴板的快捷读取交互。
- 采用 `@st.fragment` 实现局部刷新渲染。

*注：Ledom Layouter 是 Ledom 系列效率工具的核心组件之一。*
